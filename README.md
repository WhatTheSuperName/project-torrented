# Project: Torrented

**Версия:** 1.0
**Ссылка на сайт:** https://whatthesupername.github.io/project-torrented

### 📌 О проекте

**Project: Torrented** — это учебный веб-сайт, созданный для изучения современных веб-технологий. Проект демонстрирует возможность интеграции статического хостинга (GitHub Pages) с динамическим бэкендом (Firebase) для создания интерактивного сообщества.

**Технологический стек:**
*   **Frontend:** HTML, CSS, JavaScript (Vanilla)
*   **Backend & Database:** Firebase (Authentication, Firestore)
*   **3D графика:** Three.js
*   **Хостинг:** GitHub Pages

**Основной функционал:**
*   Регистрация и авторизация пользователей.
*   Разграничение прав (администратор и обычные пользователи).
*   Добавление игр (только для администратора).
*   Комментирование игр (для авторизованных пользователей).
*   Динамическая 3D сцена на заднем плане.

### ⚠️ Дисклеймер (Предупреждение)

Данный проект создан **исключительно в образовательных и портфолио-целях**.

1.  **Учебный характер:** Сайт не является коммерческим проектом и не предназначен для массового использования. Он создан для демонстрации навыков разработки.
2.  **Контент:** Ссылки на торрент-файлы или magnet-ссылки, представленные на сайте, используются только для тестирования функционала. Администрация ресурса не несет ответственности за их содержание и не поощряет нарушение авторских прав.
3.  **Ответственность:** Вся ответственность за использование материалов, найденных на сайте, лежит на посетителях. Проект не хранит на своих серверах файлы с играми, а лишь предоставляет ссылки.
4.  **Авторские права:** Мы уважаем права правообладателей. Если вы являетесь владельцем авторских прав и считаете, что размещенная информация нарушает ваши права, пожалуйста, свяжитесь с нами для её удаления.

### 🚀 Запуск проекта локально

1.  Клонируйте репозиторий: `git clone https://github.com/whatthesupername/project-torrented.git`
2.  Перейдите в папку проекта.
3.  Откройте файл `index.html` в вашем браузере (например, с помощью Live Server в VS Code).
